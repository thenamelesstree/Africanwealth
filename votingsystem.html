Polls & Voting

Voting System

The Africanwealth Voting System allows users to create polls on which other blockchain users can vote on with results and real-time stats publicly viewable, tracked and verified. Poll creators can set eligibility requirements for voter participants which gives polls fairness, transaparency, credibility and security.

Any user of the blockchain can create a poll.

Poll Creation

Any user account can create a poll and this poll can be viewed by anyone including Africanwealth non-users. However, to participate in a poll, an Africanwealth account is required and also needs to match the eligibility set by the poll creator. Every poll has configurable properties which allow poll creators to customise polls depending on the poll creators preferences, poll scope and target demograph.

Poll Properties

Poll Name

A poll can be assigned a name for easy identification and also as an indicator of the poll.

Poll Description

The description of a poll explains in detail its purpose, use case and any other information the creator may want to communicate to a viewer or poll participant.

Poll Options

A poll creator can set predefined options to which voters and participants can vote for their choice from a pre-defined set of options. Options can be any text from one-word choices to numbers or brief summaries. Also, a poll can also be open for participants to give general answers in their own words.

Poll Eligibility

Poll creators can select participants by setting conditions for which an account should meet to be considered eligible to take part in a poll. These conditions are limited to those that can be readily enforced by the blockchain i.e. makes use of Africanwealth distributed applications.
